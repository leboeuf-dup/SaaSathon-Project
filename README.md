
# Quested Webapp

Quested is a productivity-boosting webapp that helps users overcome tasks that often feel too great to tackle. It makes use of OpenRouter AI to take a list of tasks and goals from the user and turn them into a schedule of broken-down components and smaller tasks that guide the user toward their final goals in a more manageable way.

Quested gamifies the overall experience by transforming goals and smaller tasks into an RPG-like quest that the user partakes in. The app features an experience-based leveling system with regularly earnable rewards.
